# 🎓 Admission Menu System (Java)

A simple **menu-driven Java application** that helps students choose a stream based on their percentage. The program checks eligibility and recommends a suitable branch or course.

## 📌 Features

* Accepts student's percentage as input.
* Menu-driven interface using `switch`.
* Checks eligibility for:

  * Engineering
  * Medical
  * Business
* Recommends:

  * **Engineering:** CSE, Mechanical Engineering, BEE
  * **Medical:** MBBS, BDS, Pharmacy
  * **Business:** B.Com, BBA
* Displays messages for invalid choices or ineligible students.

## 🛠️ Technologies Used

* Java
* Scanner Class
* Switch Case
* If-Else Statements

## ▶️ How to Run

1. Clone or download this repository.
2. Open the project in VS Code, IntelliJ IDEA, or Eclipse.
3. Compile the program:

   ```
   javac AdmissionMenu.java
   ```
4. Run the program:

   ```
   java AdmissionMenu
   ```
